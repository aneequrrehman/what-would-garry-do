# what-would-garry-do
A Claude Code skill that channels Garry Tan when you try to log off early.

> Garry is still at his desk.
> You said you'd ship the billing flow today. It is not shipped.
> "I'll do it tomorrow" is how founders die.
> One more commit. Fix it, then you can stop.

## Installation

```
git clone https://github.com/aneequrrehman/what-would-garry-do ~/.claude/skills/what-would-garry-do
```

## Usage

`/what-would-garry-do` — or let it fire when you say things like "I'll ship tomorrow" or "good enough."
